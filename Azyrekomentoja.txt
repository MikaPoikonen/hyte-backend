--------------------------------------------------
KANSIOIDEN VAIHTAMINEN
--------------------------------------------------

Frontend projektiin:
cd ~/hyte-frontend

Backend projektiin:
cd ~/hyte-backend

Takaisin kotihakemistoon:
cd ~

Tiedostojen listaaminen:
ls


--------------------------------------------------
FRONTENDIN BUILD JA JULKAISU
--------------------------------------------------

Siirry frontend projektiin:

cd ~/hyte-frontend

Buildaa frontend (Vite):

npm run build

Poista vanha julkaistu frontend:

sudo rm -rf /var/www/html/*

Kopioi uusi build Apachelle:

sudo cp -r dist/* /var/www/html/


--------------------------------------------------
BACKENDIN KÄYNNISTYS
--------------------------------------------------

Siirry backend projektiin:

cd ~/hyte-backend

Käynnistä Node palvelin:

node src/index.js

TAI jos käytetään PM2:

pm2 start src/index.js --name hyte-backend


--------------------------------------------------
BACKENDIN UUDELLEENKÄYNNISTYS
--------------------------------------------------

pm2 restart hyte-backend

Backend prosessien tila:

pm2 status

Backend lokit:

pm2 logs hyte-backend


--------------------------------------------------
API TESTAUS PALVELIMELLA
--------------------------------------------------

Testaa että backend vastaa:

curl http://localhost:3000

Testaa API endpoint:

curl http://localhost:3000/api/omakanta/stats

--------------------------------------------------
TIETOKANTA (MariaDB)
--------------------------------------------------

Kirjaudu MariaDB:

sudo mysql -u root

Tietokannan valinta:

USE harjoitus;

Taulujen lista:

SHOW TABLES;

Taulun sisältö:

SELECT * FROM users;
SELECT * FROM dailyhealthstats;
